## 🚫 TROUBLESHOOTING

### **Issue: Button not visible**
**Solution**: Refresh page, clear browser cache

### **Issue: WhatsApp doesn't open**
**Solution**: Check if WhatsApp is installed (mobile)

### **Issue: Message not pre-filled**
**Solution**: Browser blocking, manually copy message

### **Issue: Popup won't close**
**Solution**: Click X button or tap outside popup
